# AstraCine – Integrated Cinema Operation & Management System

## 📌 Project Overview
**AstraCine** is an all-in-one cinema management platform designed to automate complex scheduling, optimize Point-of-Sale (POS) workflows, and streamline human resource management. 

This project was developed by a team of 4-5 members using **Agile/Scrum** methodology to ensure high-quality software delivery and professional collaboration.

---

## 👤 My Role: Team Leader & Fullstack Developer
As the **Team Leader**, I managed the overall project progress and technical architecture:
* **Project Management**: Utilized **Jira** to manage Backlogs, Sprints, and task distribution.
* **Workflow**: Enforced a professional **Git branching strategy** on GitHub to maintain code integrity.
* **Core Development**: Authored both Backend (Spring Boot) and Frontend (React) for mission-critical modules.

---

## 🚀 Key Features (My Contributions)
* **Intelligent Scheduling**: Developed a specialized algorithm to prevent showtime overlaps within the same cinema room.
* **Dynamic Seat Mapping**: Built a real-time system for room configuration and seat status management.
* **Integrated Sales (POS)**: Created a unified transaction system allowing customers to purchase movie tickets and snacks/popcorn (F&B) in one order.
* **E-Ticket System**: Engineered the module for automated e-ticket issuance upon successful payment.
* **Staff Management**: Currently designing a Role-Based Access Control (RBAC) module for cinema staff.

---

## 🛠 Tech Stack
* **Backend**: Java (Spring Boot 3), Spring Data JPA, Spring Security.
* **Frontend**: React, JavaScript (ES6+), Bootstrap.
* **Database**: MySQL (Optimized schema for complex entities).
* **Tools**: Jira, Git/GitHub, IntelliJ, VS Code.
